  How the bridge works:
  1. On start, C++ forks python3 python/wheels_controller.py
  2. Every step: C++ sends "step_number time interval\n" to Python stdin
  3. Python calls step(), returns "left right\n" to C++ stdout
  4. C++ calls m_acWheels->SetSpeed(left, right)

  To program the robot in Python, the user only edits the step() function in
  python/wheels_controller.py. The current default replicates the original
  behaviour (both wheels at 1000).
